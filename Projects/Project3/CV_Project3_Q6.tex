\section*{Question 6}

\begin{enumerate}
	\item The core of the \textbf{Lucas-Kanade} optical flow method involves solving the overdetermined system $\mathbf{Au} = \mathbf{b}$ for the flow vector $\mathbf{u}$ in a local window. The least-squares solution is given by:
	\[
	\mathbf{u} = (A^T A)^{-1}A^T \mathbf{b}
	\]
	Let $\mathbf{G} = A^T A$ be the $2 \times 2$ \textbf{Structure Tensor} (or Gram Matrix) of the window.\\
	Prove that a reliable and unique solution for the flow vector $\mathbf{u}$ exists \textbf{if and only if} the Structure Tensor $\mathbf{G}$ is \textbf{invertible}.
	
	\item The change in image intensity $E(u, v)$ when a window is shifted by a vector $\mathbf{d} = [u, v]^T$ is approximated using the \textbf{Second Moment Matrix M}:
	\[
	E(u, v) \approx \mathbf{d}^T \mathbf{M} \mathbf{d}
	\]
	Prove that if a region is an ideal, straight, uniform edge (not a corner), one of the eigenvalues of $\mathbf{M}$ must be zero.
	
	\item In the \textbf{Horn-Schunck} optical flow method, the smoothness term $\mathcal{E}_s$ penalizes the squared magnitude of the gradient of the flow field. For a vector $\mathbf{v} = [v_1, v_2, \dots, v_n]^T$, the squared Frobenius Norm is defined as $||\mathbf{v}||_F^2 = \sum_{i=1}^n v_i^2$.
	
	Prove that the term $\left(\frac{\partial u}{\partial x}\right)^2 + \left(\frac{\partial u}{\partial y}\right)^2$ can be expressed as the squared Frobenius Norm of the flow gradient vector $\nabla u = \left[\frac{\partial u}{\partial x}, \frac{\partial u}{\partial y}\right]^T$.
\end{enumerate}
